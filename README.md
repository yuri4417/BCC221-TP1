# 🏦 Sistema de Gerenciamento de Banco

Projeto desenvolvido em C++ para simular um sistema bancário com clientes, gerentes e transações.

---

## 📌 Funcionalidades

- Cadastrar cliente
- Cadastrar gerente
- Criar transações
- Exibir extrato
- Listar clientes e gerentes

---

## 🛠️ Tecnologias utilizadas

- C++
- STL (vector, string)

---
